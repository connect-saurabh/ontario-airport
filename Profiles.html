9 Simulation scenarios

Show code 
# Create scenarios based on engagement changes
simulation <- data_2025 %>%
  mutate(
    low_engagement = total_engagements * 0.8,
    high_engagement = total_engagements * 1.2
  )

# Predict passengers under each scenario
simulation <- simulation %>%
  mutate(
    baseline_pred = predict(model_final, newdata = data_2025),

    low_pred = predict(model_final,
      newdata = data_2025 %>%
        mutate(total_engagements = low_engagement)
    ),

    high_pred = predict(model_final,
      newdata = data_2025 %>%
        mutate(total_engagements = high_engagement)
    )
  )

11 Convert to ROI

proxy value per passenger

Show code 
 value_per_passenger <- 20

A simulation framework was developed to estimate the impact of changes in engagement levels on passenger volume. By adjusting engagement inputs and predicting passenger outcomes, the model provides a forward-looking estimate of potential ROI from community engagement strategies.

Show code 
roi_results <- simulation %>%
  mutate(
    baseline_value = baseline_pred * value_per_passenger,
    high_value = high_pred * value_per_passenger,
    incremental_value = high_value - baseline_value
  ) %>%
  summarise(
    total_incremental_value = sum(incremental_value, na.rm = TRUE)
  )

print(roi_results)

A 20% increase in engagement is associated with approximately $5.7 million in incremental passenger value over the year

Under a high-engagement scenario (+20%), the model estimates an incremental value of approximately $5.7 million, based on a conservative proxy of $20 per passenger. This reflects the potential economic impact of increased community engagement efforts.
